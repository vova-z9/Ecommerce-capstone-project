# Suitcase Shop

Responsive e-commerce website built as a capstone project to practice strict typing and modern UI development, based on a Figma layout.

🔗 https://suitcaseshop.netlify.app/

## Screenshot


![Catalog with filters](path-to-screenshot-2.png)

## Tech Stack

- TypeScript
- SASS
- HTML
- Figma (UI/UX design reference)

## Features

- Responsive layout adapting to various screen sizes and devices
- Product filtering by size, color, category, and sales, plus search and sorting
- Typed data models and application logic managed by TypeScript for improved code reliability
- Clean code architecture with a well-organized and maintainable project structure

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/vova-z9/Ecommerce-capstone-project
cd Ecommerce-capstone-project
npm install
npm run dev
```

The app will be running at `https://suitcaseshop.netlify.app/`.

## Project Structure

```
src/
├── components/     # Reusable UI elements for the e-commerce interface
├── styles/         # Modular SASS/SCSS files for styling
├── types/          # TypeScript interfaces and data model types
└── data/           # Mock data for product listings
```

## What I Learned

Building this project helped me transition from standard JavaScript to TypeScript, teaching me how strict typing catches errors early and makes application logic much more reliable. I also practiced translating a Figma design into a pixel-perfect, responsive interface while keeping my SASS stylesheets organized and easy to maintain. Next time, I plan to integrate React to handle the application's UI state more efficiently.
